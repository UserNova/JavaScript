Ce TP est de creer une application Node.js complète avec:

- Un serveur HTTP de base
- Une API RESTful
- Manipulation de fichiers
- Utilisation d'Express.js
- Gestion des variables d'environnement
- Journalisation des requêtes
- Interface utilisateur simple

  
Ce TP a permis de découvrir les modules de base de Node.js:

- http: Pour créer des serveurs web
- fs: Pour manipuler les fichiers
- path: Pour gérer les chemins de fichiers
- url: Pour analyser les URLs
  
Et des modules externes populaires:

- express: Framework web
- morgan: Middleware de journalisation
- dotenv: Gestion des variables d'environnement
